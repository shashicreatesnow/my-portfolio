# Ritam Admin Panel

> The management backbone behind the Ritam news app — a full admin panel for content, users, and engagement.

## Quick Facts

- **Client:** Ritam (news company, India)
- **Category:** UI/UX Design
- **Role:** Lead Product Designer
- **Timeline:** 4 weeks (2025)
- **Industry:** News / Media — Admin & Management System
- **Status:** Completed
- **Part of:** Ritam Product Ecosystem (App + Admin Panel + Web App)

## The Problem

Designing the admin panel of a news website is a big challenge. It involves management and creation of articles, short news, shorts. Managing access of the admin panel to different users. Managing notifications, comments, content visibility, and more — all while keeping it simple enough for non-technical editors to use daily.

## The Solution

### 1. Simple and Powerful Dashboard
- Top stat cards for key parameters (1,918 / 250 / 25 / 1,500 / 2,564)
- Graph view of downloads over time
- Top News Group breakdown
- Top Content shortcut with direct access to articles
- Header with search and profile
- Clean sidebar navigation: Dashboard, Content, Whitelisting, Engagement, Settings

### 2. Very Functional Content Page
- Three sections for three content types: Articles, Shorts, News
- Each page has similar functionality with slight differences
- Content cards in table format for easy readability
- Filters for sorting and finding content
- Important actions (Publish, Unpublish, Delete) accessible at top
- "Add New" button for creating content

### 3. UX Decisions in Content Page
- Status indicator: green dot (Published), red dot (Unpublished) — instant visual scan
- Content data in separate cards (not rows) for easy readability per item
- Filters with inbuilt search to stop manually finding options
- Actions dropdown for batch operations — select multiple cards and publish/unpublish/delete at once
- Checkbox selection on each content piece

### 4. Interactive Text Editor for New Articles
- Custom built text editor for editing articles
- Important info tokens (date, reading time)
- Rich text editing field
- Visibility actions: Publish, Preview, Save Draft
- Add header image option
- Sidebar with important options: News Group, Author's Name, Category, Content Language, Demographic Visibility, Text Index

### 5. UX Decisions in Text Editor
- Dropdown to assign categories with visual selection chips (green tags)
- Section to show all category selections and remove them if necessary
- "Add article in new language" opens a new overlay of the text editor
- Content Language setting only changes language of content in existing languages
- Demographic Visibility controls for targeted distribution

### 6. Crawler to Crawl Content from Internet
- Search for any content by tag/topic
- Crawled results shown in list format with author, date, source
- Three import options: Summarize All, Summarize Selected, Import All Summarized
- Helps editors quickly source and curate external content

### 7. Whitelisting — Content Visibility Management
- Domains list with visibility toggles (green = visible, red = hidden)
- Content cards for each domain showing URL, author, category
- Quick toggle to show/hide content sources
- Filters by category and source

### 8. Notifications Management (Engagement)
- Create new push notifications with Title, Message, Target URL, Target User
- View all previous notifications in chronological list
- Target specific user segments
- Split view: compose on left, history on right

### 9. Comments Management (Engagement)
- View all user comments across all articles
- Filter by Author, News Group, Tags
- Show/Hide toggle per comment for moderation
- Full comment text visible with user name and timestamp
- Manage comments to hide or delete them

### 10. Team & Roles (Settings)
- Team Members list with role badges and last active time
- Roles panel with permission management
- Define access and permissions per role (Super Admin, Admin, Author, Parker, Content Manager)
- Assign specific users to roles
- Edit and modify member access

### 11. Content Settings — Categories and Languages
- Manage news categories as chips (Sports, News, Trade, Arts, Adventure, Exploration, Technology, Food, Fitness, Education, Photography, Design, Fashion, Gaming, Lifestyle, Technology)
- Add/remove categories with one click
- Feed Languages management (Spanish, Marathi, Malayalam, Hindi, Romanian, Telugu, Malayalam, Bengali, Tamil, Gujarati, Marathi, Punjabi)
- Add/remove supported languages

### 12. Audit Log (Settings)
- See all activities done by any user in the admin panel
- Detail and timestamp for each activity
- Click to see full activity details per user
- User name, ID, date, time, and hours logged

## Outcomes

- Average article publish time reduced
- Stats show impactful insights
- New text editor window makes it easier to create content in many languages
- Easy to manage engagements on app
- Power to decide which user has how much access
- Editors gained more time to focus on storytelling and creative work
- New editors found the system easier to learn and use
